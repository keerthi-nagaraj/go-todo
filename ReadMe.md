# 📘 Todo REST API (Go)

A simple RESTful **Todo API built in Go**, designed to demonstrate clean project structure, modular packages, and lightweight data persistence using a JSON file — no external database required.

---

## 🧭 About

This project is a minimal yet functional **task management API** written in Go.  
It follows a clear, modular design — separating logic into **models**, **controllers**, and a simple JSON-based data layer.

The goal is to provide a clean foundation for beginners learning Go web development and REST APIs, with room to extend it (e.g., by adding databases, authentication, or configuration via `.env`).

---

## ⚙️ Features

- ✅ RESTful endpoints for task CRUD operations  
- ✅ Persistent storage via `data/tasks.json`  
- ✅ Modular structure (`models`, `controller`, `config`)  
- ✅ `.env` support for configuration  
- ✅ No external database — portable and self-contained
